# Ir, estar, y dar

## Ir (to go)

| Subject | Verb |
|:-------:|:----:|
| Yo | voy |
| Tú | vas |
| Él/Ella/Ud. | va |
| Nosotros/as | vamos |
| Vosotros/as | vais |
| Ellos/Ellas/Uds. | van |

## Estar (to be)

| Subject | Verb |
|:-------:|:----:|
| Yo | estoy |
| Tú | estás |
| Él/Ella/Ud. | está |
| Nosotros/as | estamos |
| Vosotros/as | estáis |
| Ellos/Ellas/Uds. | están |

## Dar (to give)

| Subject | Verb |
|:-------:|:----:|
| Yo | doy |
| Tú | das |
| Él/Ella/Ud. | da |
| Nosotros/as | damos |
| Vosotros/as | dais |
| Ellos/Ellas/Uds. | dan |

## Tips for using verbs

- In expression with multiple verbs, only the first is conjugated
- Other verbs remain in the infinitive
- Formula for the future: "ir (conjugated) + a + infinitive"
