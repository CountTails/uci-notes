# Pretérito

## Using the preterit

- Past tense
- Completed actions
- Time stamp
- Single action
- Sequence of actions
- Interrupting action

## Conjugating the preterit

### Conjugating -ar verbs

| Subject | Ending |
|:-------:|:----:|
| Yo | -é |
| Tú | -aste |
| Él/Ella/Ud. | -ó |
| Nosotros/as | -amos |
| Vosotros/as | -asteis |
| Ellos/Ellas/Uds. | -aron |

## Stem changes verbs

### Regular verbs

- Only attributed to -ir verbs
- Only in the third persons
- Only two stem changes
    1) o -> u
    2) e -> i
