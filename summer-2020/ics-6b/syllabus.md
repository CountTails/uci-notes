# I&C SCI 6B Syllabus

## Course description

Name: Boolean Logic and Discrete Data Structures

Units: 4 units

Summary: Relations and their properties; Boolean algebras, formal languages; finite automata.

Prerequisites: High school mathematics through trigonometry
