# El futoro

## Conjugar

- Stems consist of the entire infinitive form plus

| subject | ending |
| ---- | ---- |
| yo | -é |
| tú | -ás |
| él/ellos/Ud. | -á |
| nosotros/os | -emos |
| ellos/ellas/Uds. | -án |
